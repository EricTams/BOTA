## UI Style Guide

### Color Palette
- **Primary**: Warm wood tones (#8B4513, #A0522D)
- **Accent**: Gold (#FFD700), Brass (#B5A642)
- **Text**: Dark brown (#3E2723) on light backgrounds, White (#FFFFFF) on dark
- **Parchment**: Beige (#F5DEB3), Tan (#D2B48C)

### Typography (CSS, not assets)
- Headers: Fantasy/medieval style fonts
- Body text: Readable serif or sans-serif
- High contrast for readability

### Design Principles
- **Maritime theme**: Ropes, anchors, waves, nautical elements
- **Fantasy aesthetic**: Medieval fantasy, not modern
- **DOTA influence**: Epic, dramatic, heroic style
- **Readability first**: UI must be functional, beautiful second
- **Consistent style**: All UI elements should feel cohesive

---

## Generation Tips
1. **Transparent backgrounds** - Essential for UI overlays
2. **High contrast** - Icons must be visible on various backgrounds
3. **Simple, clear** - UI elements should be instantly recognizable
4. **Consistent style** - Use similar prompts for cohesive look
5. **Size appropriately** - Larger for logos, smaller for icons
6. **Test on dark and light** - Ensure visibility in all contexts

## Technical Notes
- UI elements are DOM-rendered (not Canvas)
- Loaded as `<img>` tags or CSS backgrounds
- Can use CSS transforms for hover effects
- Keep file sizes reasonable for web loading
- Use PNG format for transparency support
