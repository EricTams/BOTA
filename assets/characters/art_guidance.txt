# Character Asset Art Guidance

## Overview
Character portraits for captains and crew members in the maritime trading simulation.
All portraits are 1024x1024 bust shots with opaque painterly backgrounds, rendered in epic fantasy game style.

### Generation Template for Captains

```bash
# Navigate to Imagine directory first
cd Imagine

# Generation command format:
python imagine.py generate "Close-up portrait of [CHARACTER] from Dota 2, [DETAILED DESCRIPTION], [FACTION AESTHETIC]. Epic fantasy game character portrait, painted art style, [LIGHTING], [COLOR SCHEME]. Bust shot, face and shoulders visible. Painterly background with [THEME]." ../assets/characters/captain_[name].png --size 1024x1024 --quality high --background opaque

# Return to project root
cd ..
```

## Crew Portraits

### General Specifications
- **Size**: 1024x1024 pixels
- **Format**: PNG with opaque painterly backgrounds
- **Style**: Epic fantasy game character portrait, painted art style
- **Composition**: Close-up bust shot
- **Background**: Painterly backgrounds matching character theme

### Crew Types

#### Warrior Crew
- **Dice Pool**: 2 dice, heavy Red (Strength)
- **Specialization**: Physical challenges, intimidation, combat
- **Visual Style**: Armored, muscular, weapons visible
- **Example**: Thorgrim Ironbeard (Stonehall warrior)

#### Mage Crew
- **Dice Pool**: 2 dice, heavy Blue (Magic)
- **Specialization**: Arcane challenges, persuasion, mysticism
- **Visual Style**: Robes, magical energy, mystical aesthetic
- **Example**: Arcane scholars, wizards

#### Rogue Crew
- **Dice Pool**: 2 dice, heavy Green (Agility)
- **Specialization**: Stealth, navigation, trickery
- **Visual Style**: Light armor, quick appearance, sneaky aesthetic
- **Example**: Thieves, scouts, rangers

#### Sailor Crew
- **Dice Pool**: 2 dice, Green/Blue mix
- **Specialization**: Navigation, weather, water challenges
- **Visual Style**: Naval clothing, nautical equipment
- **Example**: Navigators, ship's crew

#### Merchant Crew
- **Dice Pool**: 2 dice, Blue/Green mix
- **Specialization**: Negotiation, trading, social challenges
- **Visual Style**: Fine clothing, merchant aesthetic
- **Example**: Traders, diplomats

### Crew Quality Tiers

**Novice** (50-100g):
- Simple appearance, basic equipment
- 1-2 colored faces per die

**Skilled** (200-500g):
- Professional appearance, quality equipment
- 3-4 colored faces per die

**Expert** (800-1500g):
- Veteran appearance, exceptional equipment
- 4-5 colored faces per die

**Legendary** (2000g+):
- Heroic appearance, unique/magical equipment
- 5-6 colored faces per die

### Current Crew Portraits

- Placeholder portrait for crew members
- Neutral appearance, suitable for any class
- Can be used as fallback when specific portrait not available

### Future Crew Generation

When generating specific crew portraits:
1. Follow captain portrait style (1024x1024, painted art, bust shot)
2. Match faction aesthetic if crew is faction-specific
3. Include visual cues for class (armor for warrior, robes for mage, etc.)
4. Quality tier should be visible in equipment/appearance
5. Name files as: `crew_[class]_[faction]_[number].png`
   Example: `crew_warrior_stonehall_01.png`

## Design Principles

1. **Distinctive characters**: Each portrait should have unique, memorable appearance
2. **Faction consistency**: Use faction color schemes and themes
3. **Class clarity**: Visual cues should indicate character class/role
4. **Quality visible**: Higher tier characters should look more impressive/heroic
5. **DOTA inspiration**: Reference Dota 2 character designs for style consistency
6. **Painterly style**: Not photorealistic, but detailed painted art
7. **Dramatic composition**: Epic fantasy aesthetic, heroic presentation
8. **Readable at scale**: Must be recognizable when displayed at smaller sizes
