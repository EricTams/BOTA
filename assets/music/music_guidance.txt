BOTA - Background Music Generation Guidance
============================================

This file documents the prompts used to generate background music for the game.

Generation Tool: ElevenLabs Text-to-Sound-Effects API
Command: python Imagine/imagine.py makenoise [prompt] [output] --duration [seconds]

All tracks are 30 seconds long with fade in/out, mellow atmospheric Caribbean instrumental style.

Volume Configuration: See ../sounds/volume_config.js for volume settings (all music tracks set to 0.3 / 30%)

Music System:
- Managed by MusicManager (src/music.js)
- Randomly selects tracks with 2-second fade in/out transitions
- Automatically queues next track when current one ends
- Avoids repeating the same track twice in a row
- Starts automatically when game begins playing
